# Adding action words next to result

The _return_ or _console.log()_ action words.

``` javascript 

let a = 2;
let b = 1

const trueIfGreater = (a,b) => {

if(a > b) {return 'a is greater than b'} 
else {'a NOT greater than b'}

}

// result = 'a is greater than b'
```
